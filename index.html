About this website

A docs-style Arch Linux install guide

This site explains Arch Linux like a really patient person would. Short steps. Easy words. No confusing wall of text.

Installation

1. Boot the Arch ISO

Put the USB in. Start the computer. Choose the Arch USB.

2. Check internet

This just checks if the internet is alive.

ping archlinux.org

3. Open the disk tool

This is where you make space for Arch.

cfdisk

4. Install the base system

This puts the first important Arch files onto the disk.

pacstrap /mnt base linux linux-firmware

5. Save the disk map

This tells Arch where everything lives.

genfstab -U /mnt >> /mnt/etc/fstab
