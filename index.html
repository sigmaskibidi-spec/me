Installation

1. Boot the Arch ISO

Use your USB installer and boot into the live environment.

2. Check internet

ping archlinux.org

3. Partition disk

cfdisk

4. Install base system

pacstrap /mnt base linux linux-firmware

5. Generate fstab

genfstab -U /mnt >> /mnt/etc/fstab
