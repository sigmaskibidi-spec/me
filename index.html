Installation with archinstall

This section is made for normal people, not command-line wizards. archinstall is the guided installer shipped with Arch Linux, so you follow a menu instead of doing everything by hand. 1

1. Boot the Arch ISO

Put the USB in. Start the computer. Choose the Arch USB.

2. Start the guided installer

This opens the guided installer. Use the arrow keys, Enter, and the menu. 

archinstall

3. Choose language

Pick English if you want the simplest path. This just changes the menu language.

4. Set mirrors

Pick your country or closest region. This helps downloads feel less slow.

5. Choose the disk

Select the drive you want Arch installed on. Be careful here — this is the part that can erase data.

6. Pick a partition layout

For a simple install, use the guided layout options and choose a filesystem like ext4 if offered.

7. Pick a profile

Choose a desktop profile if you want a normal graphical system, or keep it minimal if you prefer a plain setup.

8. Set user and password

Create your username, password, and any admin/root options the menu asks for.

9. Confirm and install

Review the summary first. If everything looks right, begin the install and wait for it to finish.

10. Reboot

When it says the install is done, reboot the machine and remove the USB.
