Easy Installation (archinstall)

We use the built-in installer. No hard commands.

archinstall
  • Choose language
  • Select mirrors (your country)
  • Pick your disk
  • Choose filesystem (ext4)
  • Select desktop (optional)
  • Create user + password
  • Install and wait
